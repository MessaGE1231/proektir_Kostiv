Инструкция по использованию новостного сайта "Alpha-News".
1. Откройте папку new.
2. Запустите файл index.html
Использование модальных окон
1. Наведитесь на новость
2. Если появилась анимация, нажмите на нее
3. Откроется модальное окно
4. Что-бы закрыть, нажмите Escape
Регистрация
1. В правом верхнем углу на главной странице, нажмите на кнопку "Личный кабинет"
2. Введите все данные
3. Нажмите зарегестрироваться
4. При успешной регистрации сайт оповестит вас
5. Что бы войти в аккаунт, нажмите на кнопку "Уже есть аккаунт"
6. Введите те данные, которые были при регистрации
